This project is an Excel-based dashboard that analyses the performance of different mutual fund categories across different time periods. The dashboard offers visual insights into category-wise returns, helping to identify top-performing funds within each category.

Key Features

Category-wise performance comparison (Large Cap, Mid Cap, Small Cap, Flexi Cap, Multi Cap)

Top-10 performing funds within the selected category

Heatmap view for performance consistency across time periods

Interactive slicer to switch between categories dynamically

Tech / Tools Used

Microsoft Excel (Pivot Tables, Charts & Conditional Formatting)

Data cleaning & aggregation across multiple category sheets

How to Use

Download the Excel file from this repository

Open the Dashboard sheet

Use the Category Slicer to select a fund category

Purpose

This dashboard helps analyse category behaviour over time and enables users to quickly identify which fund performs better within a category.
